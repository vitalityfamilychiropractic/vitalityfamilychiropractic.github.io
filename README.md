Public website for Vitality Family Chiropractic.

# Map to Data Files

```
_config.yml: Site-wide configuration settings
_data/:
  features.yml: Big buttons on home page (label, link, and image)
  general.yml: Name to go with "Hi, I'm ..." and clinic description on home page
  locations.yml: Location names, subsite URL root, address, phone number, email, URL to map, and hours
  nav.yml: Navigation menu items
  pricing.yml: Clinic visit pricing data
  team.yml: Team member biographies
  why-me.yml: Experience and Passions tabs on home page
contacts.md: Lead-in text on Contact page
our-values.md: Our Values page
pricing.md: Lead-in text on Pricing page
specialties/: Specialty pages
team/: Individual team member pages (should be empty below front matter)
```
